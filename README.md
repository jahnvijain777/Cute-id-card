# Electron ID Card App

A stylish Electron application that allows users to create and customize their own ID cards with personal details.

## Features

- **Splash Screen**: Animated splash screen with restart capability
- **Custom ID Cards**: Create personalized ID cards with your own details
- **Navigation**: Easy navigation between splash screen and main application
- **Responsive Design**: Beautiful dark-themed UI with neon accents
- **Edit Functionality**: Modify your ID card details anytime

## Installation

```bash
npm install
```

## Running the App

```bash
npm start
```

## How to Use

1. **Launch the App**: Run `npm start`
2. **Splash Screen**: Watch the animated splash screen (3.5 seconds)
3. **Click Continue**: Skip directly to the ID card creator
4. **Enter Your Details**: Fill in your name, age, college, and hobbies
5. **Create Your Card**: Click "Create Card" to generate your personalized ID
6. **Edit Card**: Click "Edit Card" at any time to modify your details
7. **Restart**: Click "Restart" to go back to the splash screen
